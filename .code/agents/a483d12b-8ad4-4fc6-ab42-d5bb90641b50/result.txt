```csharp

    await _verificationStateManager.IsTypeVerifiedAsync(typeRef.TypeName);

    await _verificationStateManager.IsTypeVerifiedAsync(typeRef.TypeName));
await Task.WhenAll(verificationTasks);
```

**Blocking Async in Dispose** (`McpHttpClient.cs:408-414`):
```csharp

    _httpClient?.Dispose();

{
    await DisconnectAsync();
    _httpClient?.Dispose();
}
```
